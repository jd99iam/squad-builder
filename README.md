### 조기축구 스쿼드 빌더 
조기축구 할 때 매번 일일히 스쿼드 짜기 귀찮아서 만들었습니다. 
저희 팀은 n주간 불참횟수가 적은 사람, 즉 참석율이 높은 사람을 3쿼터 뛰게 해주기 때문에 불참횟수를 입력받습니다.
